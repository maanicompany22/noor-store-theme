# 🌟 ثيم نور - Noor Store Theme

ثيم أنيق ومتطور للمتاجر الإلكترونية على منصة سلة (Salla) مع مجموعة كبيرة من المزايا والبلوكات المختلفة.

## ✨ المميزات الرئيسية

### 🎨 تصميم عصري وأنيق
- تصميم متجاوب يعمل على جميع الأجهزة
- ألوان متدرجة جميلة وتأثيرات بصرية متقدمة
- خطوط عربية احترافية (Cairo)
- دعم كامل للغة العربية والاتجاه RTL

### 🚀 بلوكات متعددة ومتنوعة
- **سلايدر رئيسي محسن** - عرض الصور والعروض بشكل جذاب
- **روابط سريعة** - وصول سريع للفئات المهمة
- **ميزات المتجر** - عرض المزايا بشكل واضح
- **فئات المنتجات** - تصفح سهل ومنظم
- **المنتجات المميزة** - عرض أفضل المنتجات
- **بنرات العروض** - عروض خاصة جذابة
- **أحدث المنتجات** - منتجات جديدة وصلت للتو
- **بنرات مربعة** - عرض الفئات بشكل جميل
- **آراء العملاء** - تعزيز الثقة في المتجر
- **النشرة البريدية** - جمع البريد الإلكتروني
- **انستقرام** - عرض منشورات التواصل الاجتماعي
- **ميزات إضافية** - معلومات إضافية عن المتجر

- خطوط متعددة

## 🛠️ التثبيت والإعداد

### المتطلبات
- Node.js 16.0.0 أو أحدث
- npm 8.0.0 أو أحدث
- حساب على منصة سلة

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone https://github.com/username/noor-store-theme.git
cd noor-store-theme
```

2. **تثبيت التبعيات**
```bash
npm install
```

3. **بناء الثيم**
```bash
# للتطوير
npm run dev

# للإنتاج
npm run build-prod
```

4. **معاينة الثيم**
```bash
npm run preview
```

5. **نشر الثيم**
```bash
npm run deploy
```

## 🎯 كيفية الاستخدام

### 1. تخصيص الألوان
يمكنك تغيير الألوان من خلال ملف `assets/css/app.css`:

```css
:root {
  --primary-color: #2563eb;    /* اللون الأساسي */
  --secondary-color: #7c3aed;  /* اللون الثانوي */
  --accent-color: #f59e0b;     /* لون التمييز */
}
```

### 2. إضافة بلوكات جديدة
يمكنك إضافة بلوكات جديدة في ملف `twilight.json`:

```json
{
  "key": "unique-key",
  "title": "عنوان البلوك",
  "icon": "sicon-icon-name",
  "path": "path.to.component",
  "fields": [...]
}
```

### 3. تخصيص الصفحة الرئيسية
يمكنك تعديل الصفحة الرئيسية من خلال ملف `src/views/pages/index.twig`.

### 4. إضافة مكونات جديدة
يمكنك إنشاء مكونات جديدة في مجلد `src/views/components/`.

## 🎨 التخصيص

### الألوان
- `--primary-color`: اللون الأساسي للثيم
- `--secondary-color`: اللون الثانوي
- `--accent-color`: لون التمييز
- `--success-color`: لون النجاح
- `--danger-color`: لون الخطر
- `--warning-color`: لون التحذير

### الخطوط
- `--font-family`: عائلة الخط (Cairo)
- `--font-size-*`: أحجام الخطوط المختلفة

### المسافات
- `--spacing-*`: المسافات المختلفة بين العناصر

### الظلال
- `--shadow-*`: أنواع الظلال المختلفة
